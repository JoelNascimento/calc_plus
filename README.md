# 📱 Calc Plus

Uma calculadora moderna com histórico de operações usando Hive e layout responsivo com Flutter.

## ✨ Funcionalidades

- Cálculo de expressões com `+`, `-`, `×`, `÷`, `√`, `()`.
- Histórico salvo localmente com Hive.
- Design responsivo.
- Armazenamento e recuperação de operações.
